# 入門の始め方

この入門は、 動画(YouTube),　文章（Qiita）， 動くコード(GitHub) の3つを提供している入門で、究極のReact入門になっています。

そのため、YouTubeかQiita記事を見ながら、このレポジトリを利用してください。

# このアプリの起動方法

```
yarn install
```

で各種パッケージをインストールしたあとに、

```
yarn start
```

で `http://localhost:3000` にアプリが立ち上がります。
